# Salio (Android)

![Salio](docs/img.png)

A fintech analysis android app that visually presents M-Pesa transactions

## Features:

* **Visual Presentation:** Present M-Pesa transactions in interactive charts
* **AI:** Custom multinomial regression for classifying transaction types

## Tech stack

- **Language:** TypeScript
- **Framework:** React Native

## Requirements

- Android SDK
- Java 21

## Setup

1. Clone the repository

```commandline
git clone https://github.com/Jim-03/Salio
cd Salio
```

2. Install the dependencies

```commandline
npm install 
```

3. Prebuild the app

```commandline
npx expo prebuild --clean
```

4. Launch

```commandline
npx expo run:android
```

> **NOTE:** Phone should be connected to your device via USB
